# Web portfolio

My personal web portfolio & blog.

## Build

Requirements:
  - Node
  - Python 3
  - Rust

Clone with `--recurse-submodules` to also get blog posts.

```sh
npm install
npm run build:pre # build submodules
npm run build
```
